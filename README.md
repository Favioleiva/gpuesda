# GPU Powered ESDA

`gpu_esda` is an independent, validated CPU/GPU numerical backend for exploratory spatial data analysis. It targets inverse-distance weights, spatial lag, Global and Local Moran statistics, conditional permutation inference, FDR, and batched variable/panel workflows while retaining automatic CPU fallback.

The project is under active MVP construction. See [PLAN.md](PLAN.md) and [STATUS.md](STATUS.md).
